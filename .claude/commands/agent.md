# React Native Supabase Development Agent

Use the `@react-native-supabase-dev` agent for specialized help with this React Native + Supabase app.

## Quick Examples

### Build Admin Features
```
@react-native-supabase-dev Create a program creation screen where admins can enter program details, add items, and save to Supabase
```

### Add Real-Time Updates
```
@react-native-supabase-dev Implement real-time subscriptions for the volunteer home screen to show live program updates
```

### Setup Notifications
```
@react-native-supabase-dev Configure push notifications to alert volunteers before service starts
```

### Create Custom Components
```
@react-native-supabase-dev Build a timeline component showing today's program items with countdown timers
```
