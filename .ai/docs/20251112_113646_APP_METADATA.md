# Play Store App Metadata

Authoritative metadata for the Sunday Program Volunteer app used in Google Play Console and EAS.

## 1. App Details

- App name: Mindnistry Volunteers
- Package name (applicationId): com.mindnistry.volunteerapp

### Short Description (≤ 80 characters)

Organize volunteers and Sunday programs with simple, clear coordination.

### Full Description (≤ 4000 characters)

Sunday Program Volunteer helps churches and ministries organize their Sunday services and volunteers with clarity and ease.

Key features:
- Create and manage Sunday program schedules in one place
- Assign volunteers to roles (e.g., worship, tech, ushers, teachers)
- Share program details and assignments with your team
- Keep everyone updated on changes in real time
- Designed to be simple, focused, and easy for non-technical users

With Sunday Program Volunteer, leaders can reduce confusion, avoid last-minute scrambling, and ensure every Sunday service is well covered and clearly communicated.
